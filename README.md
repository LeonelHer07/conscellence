# Conscellence

Conscellence es un proyecto personal para practicar **ReactJS**. Se trata de una **landing page** que muestra los servicios y capacidades que una empresa podría ofrecer a sus clientes, con un diseño limpio y moderno.


## Tecnologías Utilizadas

- ReactJS
- CSS / Tailwinds
- HTML5
- JavaScript (ES6+)

## Instalación

Clona el repositorio e instala las dependencias:

```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/conscellence.git

# Entrar a la carpeta del proyecto
cd conscellence

# Instalar dependencias
npm install

# Ejecutar la aplicación en modo desarrollo
npm start

[Ver demo](https://leonelher07.github.io/conscellence/)
